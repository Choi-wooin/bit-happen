상세페이지 관리

카드별 상세페이지 본문을 3개 섹션으로 입력합니다. 에디터에서 작성하고 필요하면 HTML 소스로 직접 수정할 수 있습니다. 이미지와 동영상 크기는 HTML에서 --detail-media-height-pc 값만 수정하면 되고, Tablet/Phone은 그 값을 비율로 따라갑니다.

개발의도 / 필요성 / 개념

서비스가 왜 필요한지, 무엇을 해결하는지, 어떤 개념인지 설명합니다.

아키텍처 / 시스템구성 / 동작원리

구성요소, 데이터 흐름, 처리 방식, 기술 스택을 중심으로 설명합니다.

레퍼런스 / 관련자료

참고 이미지, 도식, 링크, 표, 비교자료, 데모 영상 등을 정리합니다.

이미지는 jpg, jpeg, png, gif, webp, bmp, svg 형식으로 최대 10MB까지, 동영상은 mp4, webm, avi, mov, mkv 형식으로 최대 100MB까지 media.bithappen.kr:4443으로 업로드됩니다.
